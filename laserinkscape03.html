Saving keychain or any file in Inkscape for lasering

The Lotus Maxx plot 125 can only use DXF, PNG, JPEG/JPG, Bitmap files only. Epilog can use SVG, DXF, PDF, JPEG/JPG, PNG file formats.


Both laser uses different file extension so please follow the tutorial as it is made simplified for you to follow.

Saving the file for Lotus Maxx Plot 125

Step 1: Select the part which has to cut. Press Shift in the keyboard and left click in your mouse to the part you want to cut.

Step 2: Go to file and select export.

Step 3: In the export panel check the box with "Export selected only"

Step 4: Change the filename in the bottom . When the dialog box appears save the file.

Step 5: Now select only the name/text inside with different color. Same repeat the step 3. In the export panel check the box with "Export selected only"

Step 6: You must have two file one for engraving and one for cutting. Bring those file to your teacher.
