Morse Code

Morse code

A very scientfic way to send coded message. Take another paper and fill sheet with all alphabets with a morse code.

Program your Calliope mini by following the step ahead. This will help you fill the sheet. Also you can make your name morse code.

In morse code when you press button A you get a dot (.) and on button B you get a blank (_). In this combination you code your letters.

Start your Open Roberta Lab

When you login your open roberta lab login it will be blank. For the Morse Code you need 1 with square section and 2 with star sections. You need to use Message block from the 2 section.

Create a Variable

Variable is like a bowl that can store. Yes it can help you store some values like numbers, texts, boolean number that is 0 and 1 inside a varaibale. LIke in bowl we can always take our candy and eat. Same from variables you can take out variable and use them.

Set a radio channel

Only needed if you want to program the recieving side of the morse that could help you send messages. When you and your friend is programming add set to channel block. That will enable radio function of Calliope mini and in same channel you can send your friends signals.

Add infinte loop and if block

Repeat indefinte loop block will help you run your program until you forcefully stop. The if block helps you decide options between A and B button pressing.

Code for button A to print (.)

Add button A which We will add show image block to display dot. Then add Message block from 2 star and add a string inside it should have a dot (.). Play a sound block can be added for more cool feature.

Code for button B to print (_)

Add button B which We will add show image block to display blank. Then add Message block from 2 star and add a string inside it should have a blank (_). Play a sound block can be added for more cool feature.

Add wait and clear block

After the if..else block add wait block for 200ms and clear display so the memory can be cleaned for your next input.

Happy then download and run

If you are ready with your code lets try running in your Calliope mini. Save the code with name morsesend.HEX. Now take the paper and fill this sheet spell your name

Program reciever Calliope to send message in Morse code

Call your friends and start programming in his/her Open Roberta Lab for the recieving end.

Code Reciever end

Follow the code in the video and edit your code after the sender part you coded as above.

Downloading morse code reciever side

Remember to ask your friend program the reciever end and download in another calliope. Set the channel to 10 in both the morse code or else you can't be able to recieve messages.
